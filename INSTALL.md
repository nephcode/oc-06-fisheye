# INSTALL
Install `Bun` with Homebrew on Mac (By Nepha Code) For Neah
````
brew tap oven-sh/bun
brew install bun
````

Create a new file
````
{
  "name": "oc-06-fisheye",
  "module": "index.ts",
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview",
    "serve": "vite preview"
  },
  "devDependencies": {
    "@types/bun": "latest",
    "sass": "^1.77.4",
    "vite": "^5.2.12"
  },
  "peerDependencies": {
    "typescript": "^5.0.0"
  },
  "dependencies": {
    "vite": "^5.3.3"
  }
}
````

Install with `Bun`
```
bun init
bun add vite
```

```
bun create vite fisheyes --template vanilla
```


````
bun install
````


````
bun run dev
````
